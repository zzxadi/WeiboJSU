本程序是本人大学时自学Android编程的个人项目，参考并应用了网上一些信息（比如乐分享微博），本项目源代码公开且可以随意修改。

应用源文件分为两个部分，一个是应用源代码（JSU），另一个是手机安装文件。

应用源代码（JSU）安装运行：
1.安装Eclipse 3.7
2.安装ADT15
3.安装Android SDK
4.将应用源代码导入Eclipse中
5.创建Android2.3的Android虚拟机AVD
6.运行AVD
7.AVD运行成功后，点击运行应用
(注：app key和app secret已经被我删除,如果需要运行需要到新浪微博开放平台注册应用申请app key和app secret,并修改..\JSU\src\com.jsu.utils.ConstantsUtils.java文件)

手机安装文件安装运行：
1.联接Android手机（Android系统要2.2版本以上）
2.安装JSU.apk文件
3.点击运行应用
